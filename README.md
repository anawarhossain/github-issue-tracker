
- 1️⃣ What is the difference between var, let, and const?
Answer: var, let and const is type of variable declaration in javascript. difference:
 1. var variable: This is old type variable declaration and now modarn javascript not use this.
 2. let variable: This type variable's value can be changed next time.
 3. const variable: This type variable's value can not be changed next time.

- 3️⃣ What is the difference between map(), filter(), and forEach()?
Answer: Difference: 
01. map() transforms each element and return a new array.
02. filter() test a conditon and return a new array with only the element that pass test.
03. forEach() executes a function on each element of an array, but does not return anything.
